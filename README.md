# CYPHERKIT ⚡

> Advanced Cybersecurity Toolkit Interface  
> Built for security enthusiasts, ethical hackers and developers.

---

## 📌 Overview

CYPHERKIT is a futuristic web-based cybersecurity utility toolkit designed with a modern cyberpunk-inspired interface.  
The project focuses on lightweight client-side utilities for encoding, decoding, hash identification and password analysis.

The interface is fully responsive and runs directly inside the browser without requiring any backend services.

---

## 🚀 Core Features

### 🔐 Multi-Format Encoder / Decoder
Supports multiple transformation formats including:

- Base64
- URL Encoding
- Hexadecimal
- Binary
- ROT13
- ASCII Encoding

---

### 🔍 Hash Identifier
Analyze and identify common hash formats such as:

- MD5
- SHA1
- SHA256
- SHA512

based on hash length and pattern recognition.

---

### 🛡 Password Strength Analyzer
Evaluate password security using:

- Length analysis
- Uppercase & lowercase detection
- Numeric validation
- Special character validation

with visual strength indicators.
