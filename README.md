# CYPHERKIT ⚡

> Advanced Cybersecurity Utility Toolkit

CYPHERKIT is a modern browser-based cybersecurity toolkit designed for developers, security researchers, and ethical hacking enthusiasts.  
It provides lightweight client-side utilities for encoding, decoding, hash analysis, and password security evaluation through a futuristic cyberpunk-inspired interface.

---

## ✨ Features

### 🔐 Encoder & Decoder
Supports multiple encoding and transformation formats:

- Base64
- URL Encoding
- Hexadecimal
- Binary
- ROT13
- ASCII

### 🔍 Hash Identifier
Detect and identify common hash types using pattern and length analysis:

- MD5
- SHA1
- SHA256
- SHA512

### 🛡 Password Strength Analyzer
Analyze password security based on:

- Character length
- Uppercase & lowercase usage
- Numeric characters
- Special symbols
